# 🎨 Color Mixture

A modern **Gradient Generator** built with **React** and **TypeScript** that lets users create beautiful CSS gradients in real time with a live preview and one-click CSS copy.

## ✨ Features

- Linear & Radial gradients
- Multiple color stops
- Live gradient preview
- Copy CSS instantly
- Toast notifications

## 🛠️ Tech Stack

- React
- TypeScript
- TailwindCSS
- React Toastify

## 🚀 Getting Started

```bash
git clone https://github.com/Dev-Jashjeet/Color-Mixture.git

cd color-mixture

npm install

npm run dev
```
